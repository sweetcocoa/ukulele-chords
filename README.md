# 🎸 우쿨렐레 코드 시각화

우쿨렐레 코드를 쉽게 찾고 시각화하는 웹 애플리케이션입니다.

## ✨ 주요 기능

- **코드 선택**: 12개 루트음과 8가지 코드 타입 지원
  - Major, Minor, 7th, Major 7th, Minor 7th, Diminished, Augmented, Suspended 4th

- **운지법 시각화**:
  - 직관적인 프렛보드 다이어그램
  - 각 줄에서 연주되는 음 표시 (C~B)
  - 오픈 스트링과 프렛 위치 명확히 구분

- **코드 시퀀스 관리**:
  - 선택한 코드를 시퀀스로 누적
  - 코드 삽입 및 삭제 기능
  - 전체 시퀀스를 한 화면에서 확인

- **대안 코드 지원**:
  - 우쿨렐레에서 구현이 어려운 코드는 대안 운지법 제시
  - 빠진 구성음 표시로 이론과 실제 차이 파악 가능

- **이미지 내보내기**:
  - 전체 코드 시퀀스를 PNG 이미지로 export
  - 고해상도 저장 지원

- **모바일 최적화**:
  - 반응형 디자인으로 모든 기기에서 사용 가능
  - 터치 인터페이스 최적화
  - 현대적이고 깔끔한 UI

## 🚀 사용 방법

1. **코드 선택하기**
   - Root(근음) 버튼에서 원하는 음 선택
   - Type(코드 타입) 버튼에서 원하는 타입 선택
   - 미리보기에서 선택한 코드 확인

2. **코드 추가하기**
   - "추가" 버튼을 클릭하여 시퀀스에 추가
   - 추가된 코드는 아래 시퀀스 영역에 표시

3. **시퀀스 관리하기**
   - "+ 삽입" 버튼: 해당 코드 다음에 새 코드 삽입
   - "✕ 삭제" 버튼: 코드 삭제
   - "전체 삭제" 버튼: 모든 코드 삭제

4. **이미지로 저장하기**
   - "이미지로 내보내기" 버튼 클릭
   - PNG 파일로 다운로드

## 🎵 지원하는 코드

### 루트음 (12개)
C, C#, D, D#, E, F, F#, G, G#, A, A#, B

### 코드 타입 (8가지)
- **Major**: 메이저 코드 (밝은 느낌)
- **Minor**: 마이너 코드 (어두운 느낌)
- **7th**: 도미넌트 7th 코드 (재즈, 블루스)
- **Major 7th**: 메이저 7th 코드 (세련된 느낌)
- **Minor 7th**: 마이너 7th 코드 (부드러운 느낌)
- **Diminished**: 디미니시드 코드 (긴장감)
- **Augmented**: 어그먼티드 코드 (불안정한 느낌)
- **Suspended 4th**: 서스펜디드 4th 코드 (부유하는 느낌)

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 현대적 스타일링, Flexbox, 반응형 디자인
- **JavaScript (ES6+)**: 바닐라 JS로 구현
- **SVG**: 코드 다이어그램 렌더링
- **html2canvas**: 이미지 export 기능
